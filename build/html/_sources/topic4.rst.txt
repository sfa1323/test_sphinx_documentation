Creating Lists
*******************

=================
Ordered List
=================
This is an ordered list:
  #. Step 1.
  #. Step 2.
  #. Step 3.

..  note:: Ordered lists usually use numerals. Nested ordered lists (ordered lists inside other ordered lists) use letters.

==========================
Unordered (Bulleted) List
==========================
This is an unordered (bulleted) list:
  * Item 1.
  * Item 2.
  * Item 3.
