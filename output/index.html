Welcome to test PrIME20 documentation!#

PrIME 20 documentation provides a variety of resources that will help you learn how to use the application, click the below panes to learn more about the particular topic.

If you don’t have PrIME installed and want to get started, follow the Installation guide.

If you are looking for a summary of its features and interface, check out the Quick Start guide.

If you are looking for information about subscriptions, go to Subscriptions.

If you are looking for information about solutions and assets, go to Solutions and Assets.

If you are looking for information about reports, go to Reporting.

If you are looking for information about managing users, go to User Management.

If you’ve run into a PrIME problem and need help solving it, take a look at our troubleshooting guide.

If you have a question about PrIME, visit the FAQs section.
